# Ridgefront Global Website - Version 1.0

This is a simple one-page website designed for GitHub + Vercel.

## Upload to GitHub
1. Open your GitHub repository.
2. Delete the old mixed-up files.
3. Upload every file from this folder, not the ZIP itself.
4. Commit changes.

## Files that must be in the main/root folder
- index.html
- styles.css
- script.js
- logo.png
- ta-ron.jpg
- tamar.jpg
- terell.jpg
- tashaun.jpg
- tyler.jpg
- tyree.jpg

Do not create an assets folder for this version.

## Deploy
Vercel will automatically redeploy after GitHub updates.
