# Ridgefront Global Website A simple responsive static website for Ridgefront Global. ## Pages - Home - Services - Meet Our Team - Contact ## Free hosting path 1. Create a free GitHub account. 2. Create a new public repository named `ridgefront-global-site`. 3. Upload all files from this folder. 4. Create a free Vercel account and connect GitHub. 5. Import the repository into Vercel. 6. After Vercel gives you a live site, add `www.ridgefrontglobal.com` in Vercel project settings. 7. In Squarespace DNS, point the domain records to Vercel using the records Vercel provides. ## Contact form note The form currently uses a simple mailto action. For a smoother form that submits without opening an email app, connect a free form service such as Formspree, Basin, or Vercel forms later.
